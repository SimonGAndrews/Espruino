# RP2040 Target Docs

This folder contains RP2040 target documentation that is derived directly from
the live Espruino codebase.

It is intended for code-adjacent material such as:

- generated interface coverage
- target-local implementation notes
- other documentation that should stay close to the RP2040 target sources

Current active target-local docs include:

- `jshardware_function_matrix_V2.md` for the human-readable RP2040 coverage view
- `jshardware_function_matrix.csv` for the raw matrix export

Archived raw snapshots live under:

- `archive/`

The active design plan and progress history live in the separate project
documentation repo:

- `https://github.com/SimonGAndrews/espruino-rp2040`
